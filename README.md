# Getting Started with Create React App

## 리액트 설치

npx create-reactapp ./

## node-sass 설치

npm install node-sass

## router 설치(주소 설정)

npm install react-router-dom

## axios 설치(api 편리한 사용)

npm install axios

## gsap 설치

npm install gsap
